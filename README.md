## 📖 Overview
This repository serves as my **Master Study Database** for the OffSec PEN-200 (OSCP) course. It contains my personal notes, methodology, checklists, and reference materials curated to master penetration testing, privilege escalation, and active directory exploitation.

## 🎯 Methodology: "Try Harder"
The core philosophy of this database is based on the **OffSec Try Harder mindset**:
- **Enumerate More:** Most OSCP failures result from incomplete reconnaissance.
- **Methodology > Tools:** Tools are secondary; a solid process is primary.
- **Document Everything:** If it isn't documented, it never happened.

## 🗂️ Database Structure
The notes are organized to follow the PEN-200 modules:
- 🔍 **Reconnaissance:** Passive & Active techniques.
- 🌐 **Web Attacks:** XSS, SQLi, LFI, and more.
- 🏢 **Active Directory:** Enumeration, Kerberoasting, and Lateral Movement.
- ⚙️ **Privilege Escalation:** Linux & Windows vectors.
- 📝 **Reporting:** Templates and best practices for OSCP-style reports.
