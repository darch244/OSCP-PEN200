##  Project Overview
This repository serves as my **Master Study Database** for the OffSec PEN-200 (OSCP) course. It documents my personal notes, methodology, checklists, and reference materials curated to master penetration testing, privilege escalation, and active directory exploitation.

---

##  Core Methodology: The DarcHacker Approach
- **Enumerate More:** Most OSCP failures result from incomplete reconnaissance.
- **Methodology > Tools:** Tools are secondary; a solid process is primary.
- **Documentation:** If it isn't documented, it never happened.
